Allowed the API pod to run against a read-only `/ref` volume.
Added a `REF_READ_ONLY_DATABASE` setting that opens SQLite via the
URI form `mode=ro&immutable=1` so no journal/WAL sidecar is required,
bumped `climate-ref` to 0.13.0 so the bundled alembic scripts match
the migrations applied by the `climate-ref-aft` chart, and replaced
the hand-rolled revision check with a status check modelled on
`ref db status` that tolerates DBs stamped by a newer CLI. The
Dockerfile now fails fast if `climate_ref_core/pycmec/*.yaml`
package data is missing from the installed wheel.
